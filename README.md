# Análise de Matriz de Confusão e Métricas de Classificação

Este projeto implementa uma análise completa de classificação binária, incluindo geração de matriz de confusão, cálculo de métricas de desempenho e visualização da curva ROC. O código foi desenvolvido para ser executado no Google Colab.

## 📋 Funcionalidades

- **Geração de Matriz de Confusão**: Cria e visualiza a matriz de confusão
- **Extração de Métricas**: Calcula VP, VN, FP, FN a partir da matriz
- **Cálculo de Métricas de Desempenho**:
  - Acurácia
  - Sensibilidade (Recall)
  - Especificidade
  - Precisão
  - F1-Score
- **Curva ROC**: Gera e plota a curva ROC com AUC
- **Análise Detalhada**: Visualizações e tabelas com todas as métricas

## 🚀 Como Executar

### No Google Colab:

1. Abra o [Google Colab](https://colab.research.google.com)
2. Crie um novo notebook
3. Copie e cole o código fornecido
4. Execute todas as células
